# UI Style Guidelines — Educational App Landing Page

## 1. Design Principles
- **Clarity First:** Content should be easy to scan and understand within seconds.
- **Friendly & Trustworthy:** The UI should feel welcoming, modern, and credible for learners, parents, and educators.
- **Educational Focus:** Visual hierarchy must support learning outcomes, features, and value propositions.
- **Consistency:** Use a repeatable visual system across sections for cohesion.
- **Accessibility:** Ensure sufficient contrast, readable typography, and inclusive interaction patterns.

---

## 2. Color System

### Primary Color
- **Primary Yellow**
  - RGB: `rgb(247, 169, 19)`
  - Usage:
    - Primary CTAs (e.g., “Get Started”, “Start Learning”)
    - Key highlights and accents
    - Active states (buttons, tabs, links)
  - Notes:
    - Avoid large background fills; use primarily as an accent or CTA color.
    - Pair with dark text for maximum contrast.

### Secondary Color
- **Deep Night**
  - RGB: `rgb(17, 17, 24)`
  - Usage:
    - Main background color
    - Navigation bar
    - Footer
    - Headings and primary text on light backgrounds
  - Notes:
    - Establishes seriousness and trust.
    - Acts as the visual anchor of the brand.

### Supporting Neutrals
- **White:** `#FFFFFF`
  - Main content backgrounds
  - Card surfaces
- **Light Gray:** `#F5F5F7`
  - Section separators
  - Backgrounds for feature blocks
- **Muted Gray:** `#9CA3AF`
  - Secondary text
  - Helper labels and captions

---

## 3. Typography

### Font Families
- **Primary Font (Headings):**
  - Sans-serif, modern, friendly (e.g., Inter, Poppins, or SF Pro)
- **Secondary Font (Body):**
  - Same family or a highly readable sans-serif for consistency

### Type Scale
- **H1:** 48–56px, Bold  
  - Hero headlines
- **H2:** 36–40px, Semi-Bold  
  - Section titles
- **H3:** 24–28px, Semi-Bold  
  - Feature titles
- **Body Large:** 18–20px, Regular  
  - Intro paragraphs
- **Body Base:** 16px, Regular  
  - Standard content
- **Caption / Meta:** 13–14px, Medium  
  - Labels, helper text

### Line Height
- Headings: 1.2–1.3  
- Body text: 1.5–1.7  

---

## 4. Layout & Spacing

### Grid System
- 12-column grid (desktop)
- 8-column grid (tablet)
- 4-column grid (mobile)

### Spacing Scale
Use an 8px-based system:
- XS: 8px  
- S: 16px  
- M: 24px  
- L: 40px  
- XL: 64px  

### Section Structure
- Clear separation between sections
- Generous vertical padding (64–96px on desktop)
- Content width max: 1100–1200px

---

## 5. Buttons & CTAs

### Primary Button
- Background: `rgb(247, 169, 19)`
- Text: `rgb(17, 17, 24)`
- Border Radius: 10–12px
- Font Weight: Semi-Bold
- Hover: Slight darken of yellow or subtle scale-up
- Shadow: Soft, warm shadow for depth

### Secondary Button
- Background: Transparent
- Border: 1–2px solid `rgb(247, 169, 19)`
- Text: `rgb(247, 169, 19)`
- Hover: Yellow background with dark text

---

## 6. Cards & Surfaces

- Background: White
- Border Radius: 16px
- Shadow:
  - Soft elevation (e.g., 0 10px 30px rgba(0,0,0,0.08))
- Padding: 24–32px
- Use for:
  - Features
  - Testimonials
  - Pricing
  - Course previews

---

## 7. Navigation

- Background: `rgb(17, 17, 24)`
- Logo: White or Primary Yellow
- Nav Links:
  - Default: White
  - Hover/Active: `rgb(247, 169, 19)`
- CTA Button in Nav:
  - Primary Button style

---

## 8. Iconography & Illustrations

- Style: Rounded, friendly, minimal
- Line or filled icons with consistent stroke width
- Accent icons using primary yellow sparingly
- Illustrations:
  - Education-themed
  - Soft gradients
  - Simple shapes and characters

---

## 9. Imagery

- High-quality, optimistic visuals
- Diverse learners and educators
- Avoid overly corporate or stock-heavy visuals
- Use subtle overlays with dark background when placing text over images

---

## 10. Motion & Interaction

- Transitions: 200–300ms ease-out
- Hover effects:
  - Buttons: scale (1.03) or glow
  - Cards: slight lift + shadow increase
- Scroll animations:
  - Subtle fade-in or slide-up
  - Never distracting or playful to excess

---

## 11. Accessibility Guidelines

- Contrast ratio: Minimum WCAG AA
- Buttons and links clearly distinguishable
- Font sizes never below 14px
- Clear focus states for keyboard navigation

---

## 12. Overall Tone & Feel

- **Smart but Friendly**
- **Motivational**
- **Modern Educational**
- **Trust-building and optimistic**

This UI should instantly communicate that the app is **effective, enjoyable, and designed for real learning success**.
